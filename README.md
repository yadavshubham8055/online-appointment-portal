Online Appointment Management Portal

A mini web application for managing doctor appointments.

Features
Display doctor list (name, specialization, consultation fee)
Book appointments
Form validation
Store appointment data in browser localStorage
Basic navigation between pages
Tech Stack
React
JavaScript
Vite
How to Run
1. Clone the repository
git clone https://github.com/yadavshubham8055/online-appointment-portal.git
2. Open project folder
cd online-appointment-portal
3. Install dependencies
npm install
4. Start the development server
npm run dev
5. Open in browser

Open the local URL shown in the terminal (usually http://localhost:5173).

Usage
View the list of doctors
Click Book Appointment
Enter patient name, date, and time
Submit the form
Appointment is saved in browser storage
Time Spent

Approximately 4–5 hours
